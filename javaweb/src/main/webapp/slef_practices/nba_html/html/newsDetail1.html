字母哥：若成联盟门面必须美国化 那我拒绝

首页新闻 NBA官网 Tree2019-03-21 16:08



北京时间3月20日，据密尔沃基哨兵报报道，扬尼斯-安特托昆博接受采访聊到了关于自己能否成为联盟门面的问题。

“很多人说我可以成为联盟的门面，”字母哥说，“最近，一些人告诉我——我身边那些最亲近的人，我的家人啊，女朋友啊， 我的妈妈，我的兄弟，他们说我没法成为联盟的门面，因为我不是美国人，我体内没有美国文化。”

之后字母哥说：“我坐下来想了想这件事，我应该更有美国范一点么，我应该进球馆时穿得更潮一点么，我应该更美国化一点 么？哥们，如果我不能在成为联盟门面时做自己，那我就不想成为联盟门面。”
